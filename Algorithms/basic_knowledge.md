| 记号       | 含义                   |
| ---------- | ---------------------- |
| Theta(N^2) | c1N^2 <= f(n) <= c2N^2 |
| O(N^2)     | f(n) <=cN^2            |
| Omega(N^2) | cN^2 <= f(N)           |
